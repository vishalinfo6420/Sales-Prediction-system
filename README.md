# 📈 Sales Prediction System using Machine Learning

A Machine Learning-based web application that predicts product sales based on advertising expenditure across **TV**, **Radio**, and **Newspaper** marketing channels. The project uses **Linear Regression** to analyze advertising effectiveness and forecast future sales, enabling businesses to make informed marketing decisions.

The application is developed using **Python**, **Scikit-learn**, and **Streamlit**, providing an interactive interface where users can enter advertising budgets and instantly receive predicted sales results.

## 📂 Dataset Information

The dataset contains advertising expenditure across different marketing platforms and corresponding product sales.

| Feature | Description |
|----------|-------------|
| TV | Advertising budget spent on TV |
| Radio | Advertising budget spent on Radio |
| Newspaper | Advertising budget spent on Newspaper |
| Sales | Product sales (Target Variable) |

## 🤖 Machine Learning Model

## 📏 Model Evaluation

The model performance was evaluated using:

- R² Score

- Root Mean Squared Error (RMSE)

These evaluation metrics measure prediction accuracy and overall model performance.

## 📈 Business Insights

- TV advertising has a significant positive impact on product sales.
- Radio advertising strongly influences sales performance.
- Newspaper advertising has comparatively less impact on sales.
- Businesses can optimize advertising budgets using the model's predictions.
- The system helps estimate future sales before launching marketing campaigns.
